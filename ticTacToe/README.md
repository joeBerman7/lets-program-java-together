# TicTacToe ❌⭕ 

A simple **Tic-Tac-Toe** game for two players, implemented with Java Swing.

## Features:
- Playable on a 3x3 grid.
- Interactive game loop with player input.
- Basic AI (optional) for single-player mode.

## Technologies Used:
- Java Swing
