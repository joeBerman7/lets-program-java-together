# SnakeGame 🐍

A classic **Snake Game** where the player controls a snake to eat food and grow longer without hitting the walls or itself.

## Features:
- Score tracking and game-over conditions.
- Adjustable speed settings.
- Restart option.

## Technologies Used:
- Java
